**进行单元测试的时候，清除数据操作是放在开始还是结束呢？**

1. 放在开始更好点，因为如果放在结束，一旦执行的程序中间出现了异常，就可能导致，清理代码没有被执行到

2. 清理操作放在开始，当执行完一个操作之后，此处 redis 里仍然是有数据的，方便我们手工检查数据
