# 一\. Set 类型基本介绍

谈到一个术语，这个术语很可能有多种含义

**Set**

1. 集合

2. 设置（和 get 相对应）

集合就是把一些有关联的数据放到一起\~

1. 集合中的元素是无序的（顺序不重要的意思）

此处说的无序是和前面list这里的有序对应的

有序：顺序很重要，交换一下顺序，就是不同的list了

无序：顺序不重要，变化一下顺序，集合还是那个集合
